# 1. Balance Brackets


The project contains a function(CheckParentesis) that uses a sequence of brackets as input and, if the order of the parentheses are valid.
This project was generated with [Java SE] version 8

```
Returns
(){}[] is valid = true
[{()}](){} is valid = true
[]{() is not valid = false
[{)] is not valid = false
```
